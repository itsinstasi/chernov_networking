Вчера провел один из первых вебинаров про нетворкинг на английском.

Вчера провел один из первых вебинаров про нетворкинг на английском. До этого делал пару лекций по скайпу для New York University, а в этот раз позвали выступить для Schneider Electric.

Задача была в том, чтобы найти общие подходы в нетворкинге для разных культур, так как вебинар был для EMEA региона - присутствовали Эмираты, Индия, Турция, Пакистан, Саудовская Аравия. Вроде получилось. С одной стороны - в некоторых из этих стран например сильна кастовость, с другой стороны работа в интернациональной компании сглаживает различия и облегчает коммуникацию) 🌍 Вспомнил свое P&G прошлое)👔
